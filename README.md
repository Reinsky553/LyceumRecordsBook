# 📖 Книга рекордов ГКЛ: Руководство по развертыванию

Этот проект представляет собой интерактивную книгу рекордов лицея. Ниже приведена инструкция по локальному запуску.

---

## 🛠 Предварительная настройка

Прежде чем запустить сервер, необходимо настроить учетные данные администратора и ключи безопасности.

### Шаг 1: Генерация защищенного пароля
Для обеспечения безопасности мы не храним пароли в открытом виде. Используйте встроенную утилиту для создания хеша:
1. Запустите скрипт генерации:
   ```bash
   python generate_hash.py
2. Введите желаемый пароль в консоль.
3. Скопируйте полученную строку (хеш) — она понадобится на следующем этапе.

### Шаг 2: Конфигурация приватных данных
Создайте в корневой директории проекта файл **private.py**. Этот файл используется для хранения чувствительной информации и не должен попадать в публичные репозитории.

Добавьте в него следующие переменные:

    # Логин для входа в панель управления
    ADMIN_USER = "ваш_логин"

    # Хеш пароля, полученный в Шаге 1
    ADMIN_PASS_HASH = "вставьте_сюда_хеш"

    # Секретный ключ для подписи сессий
    SECRET_KEY = "ваш_уникальный_секретный_ключ"

## 🚀 Запуск приложения

После того как конфигурация завершена, можно приступать к запуску сервера:

1. Активация сервера:
В терминале выполните команду:
   ```bash
    python app.py

2. Доступ к интерфейсу:
Откройте любой современный браузер и перейдите по адресу:

    http://127.0.0.1:5000
